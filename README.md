# 小红书文字转图片生成器

纯静态网页工具，打开 `index.html` 即可使用。文案在浏览器本地渲染到 Canvas，可导出 PNG。

```text
https://andycr0w.github.io/tool_xiaohongshu-card-generator/
```

## 功能

- 支持 3:4、4:5、1:1、2:3、9:16、16:9、长图七种比例。
- 支持封面标题、清单笔记、金句卡片、步骤教程、杂志封面、打勾清单、对比卡片、时间线、便签拼贴、代码摘录十种模板。
- 支持多页文案：用空行或单独一行 `---` 分页。
- 支持字体气质、字号、标题比例、行距、留白、圆角、角标、装饰、页码和自定义颜色。
- 支持一键下载当前页 PNG 或全部分页图片。
