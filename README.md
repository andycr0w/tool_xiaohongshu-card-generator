# 小红书文字转图片生成器

纯静态网页工具，打开 `index.html` 即可使用。文案在浏览器本地渲染到 Canvas，可导出 PNG。

## 部署到 GitHub Pages

1. 把 `tool_xiaohongshu-card-generator` 目录提交到你的 GitHub Pages 仓库。
2. 访问 `https://你的用户名.github.io/tool_xiaohongshu-card-generator/`。
3. 如果使用现有 `andycr0w.github.io` 仓库，推送后地址通常是：

```text
https://andycr0w.github.io/tool_xiaohongshu-card-generator/
```

## 功能

- 支持 3:4、4:5、1:1 三种比例。
- 支持封面标题、清单笔记、金句卡片、步骤教程四种模板。
- 支持多页文案：用空行或单独一行 `---` 分页。
- 支持一键下载当前页 PNG。
